# 互動規則

策展 X 互動推薦的規則。這些規則在策展步驟中引導 agent 選擇要互動的推文及使用的動作類型。

草稿品質由共用資源管控：`post-rules.md`（禁止模式、品質要求）和 `social-style-guide.md`（Persona、Voice、Good/Bad Examples）。

## 策展標準

### 值得互動

- 分享與你的興趣相關的真實洞見或經驗的推文
- 你的觀點能增添價值的討論（不只是附和）
- 你追蹤的帳號發布的、能激發靈感的內容
- 與你近期文章或貼文有關聯的推文

### 跳過

- 純新聞轉發，沒有原創評論
- 推廣性質的推文或產品發布（除非直接相關）
- 釣怒氣或製造爭議的推文
- 沒有互動的推文（0 讚、0 回覆）——可能曝光度很低

## 動作類型選擇

| 訊號 | 建議動作 |
|------|---------|
| 推文表達了你有想法的觀點 | reply |
| 推文與你寫過的東西有關 | quote |
| 推文很精彩，本身就夠完整 | retweet |
| 推文不錯但不需要回應 | like |
| 推文激發了一個獨立的想法 | post |

## 數量指引

每日執行推薦：
- 1-2 則 reply 或 quote 動作（最高價值，需撰寫草稿）
- 0-1 則獨立貼文靈感
- 0-2 則 retweet 或 like 建議
- 合計：每日 2-5 則推薦（品質重於數量）
